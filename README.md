Classificação de Atendimento Hospitalar com Machine Learning

O repositório classificando_atendimento_hospitalar apresenta um projeto de Ciência de Dados voltado para a classificação de atendimentos hospitalares utilizando técnicas de Machine Learning.

O objetivo do projeto é analisar dados médicos e auxiliar na categorização de atendimentos com base nas informações dos pacientes, contribuindo para apoio à tomada de decisão e organização de processos na área da saúde.

A aplicação foi desenvolvida utilizando Python e bibliotecas de análise de dados e aprendizado de máquina para:

* Tratamento e limpeza de dados
* Análise exploratória
* Treinamento de modelos de Machine Learning
* Avaliação de desempenho dos modelos
* Predição e classificação de atendimentos hospitalares

Tecnologias utilizadas

* Python
* Pandas
* NumPy
* Scikit-Learn
* Flask
* Machine Learning

Funcionalidades

* Classificação automática de atendimentos
* Processamento de dados médicos
* Treinamento e avaliação de modelos
* Interface web para interação com o sistema

Como executar o projeto localmente

1. Clone o repositório

git clone https://github.com/maryreginaf15/classificando_atendimento_hospitalar.git

2. Acesse a pasta do projeto

cd classificando_atendimento_hospitalar

3. Crie um ambiente virtual (opcional, mas recomendado)

Windows

python -m venv venv
venv\Scripts\activate

Linux/Mac

python3 -m venv venv
source venv/bin/activate

4. Instale as dependências

pip install -r requirements.txt

5. Execute a aplicação

python app.py

Caso o arquivo principal tenha outro nome, substitua app.py pelo arquivo correto.

6. Acesse no navegador

Após executar o projeto, abra o navegador e acesse:

http://127.0.0.1:5000

Objetivo do projeto

Este projeto foi desenvolvido como prática de aplicação de Machine Learning em um contexto real, unindo análise de dados, desenvolvimento em Python e construção de soluções inteligentes voltadas para a área da saúde.
